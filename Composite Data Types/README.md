## Composite Data Types
### Arrays 
- A sequence of elements of a single type. __Length :__ No. of Elements.
- Using Arrays
  ```
    var arr[10] int // Declaring an Array
    arr[0] = 10 // setting value for arr[0] to 10
    fmt.Println(arr[0])
    fmt.Println(arr) 
  ```
